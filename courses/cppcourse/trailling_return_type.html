C++ Trailing Return Types

In C++, the trailing return type allows you to specify the return type of a function after the parameters. This feature is useful when the return type depends on the function's parameters, especially in template programming.

Syntax:


#include <iostream>

// Function declaration using trailing return type
auto add(int a, int b) -> int {
    return a + b;
}

int main() {
    std::cout << "Sum: " << add(5, 3) << std::endl;
    return 0;
}

Key Points:

  • Trailing return type uses auto for the return type, followed by -> and the actual type.
  • It's particularly useful in cases like template functions where the return type may depend on the input types.

Example with Template:


#include <iostream>

template <typename T>
auto multiply(T a, T b) -> decltype(a * b) {
    return a * b;
}

int main() {
    std::cout << "Product: " << multiply(2.5, 3.5) << std::endl;
    return 0;
}

In the example, the return type of multiply is deduced based on the multiplication of the two input values.
