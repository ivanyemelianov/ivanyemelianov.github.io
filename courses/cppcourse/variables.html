Variables in C++

In C++, a variable is a storage location with a specific type that holds a value. Each variable must be declared before use.

Declaring Variables

Variables in C++ are declared using a type followed by a name:

int age;
double price;
char grade;

Initializing Variables

Variables can be assigned values at the time of declaration:

int age = 25;
double price = 99.99;
char grade = 'A';

Types of Variables

  • int: Integer values (e.g., int x = 10;)
  • double: Floating-point values (e.g., double y = 3.14;)
  • char: Single character (e.g., char letter = 'C';)
  • bool: Boolean values (true or false)
  • string (C++11+): Sequence of characters (e.g., string name = "John";)

Variable Scope

Variables have different scopes:

  • Local variables: Declared inside a function, accessible only within that function.
  • Global variables: Declared outside functions, accessible throughout the program.

Example Program

#include <iostream>
using namespace std;

int main() {
    int age = 30;
    double height = 5.9;
    cout << "Age: " << age << "\n";
    cout << "Height: " << height << " feet\n";
    return 0;
}

This program declares and initializes two variables, then prints their values.
