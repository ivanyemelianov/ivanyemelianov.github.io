Namespaces in C++

Namespaces in C++ help organize code and prevent name conflicts, especially in large projects or when using multiple libraries.

Declaring a Namespace

To create a namespace, use the namespace keyword:

namespace MyNamespace {
    int value = 42;
}

Accessing Namespace Members

Use the scope resolution operator :: to access elements inside a namespace:

#include <iostream>

namespace MyNamespace {
    int value = 42;
}

int main() {
    std::cout << MyNamespace::value << std::endl;
    return 0;
}

The using Directive

The using directive allows you to avoid prefixing every member with the namespace name:

using namespace MyNamespace;
std::cout << value << std::endl; // No need for MyNamespace::

However, using this in global scope can lead to name conflicts.

Nested Namespaces

Namespaces can be nested inside each other:

namespace Outer {
    namespace Inner {
        int num = 10;
    }
}

std::cout << Outer::Inner::num << std::endl;

Example Program

#include <iostream>
using namespace std;

namespace MyNamespace {
    void greet() {
        cout << "Hello from MyNamespace!" << endl;
    }
}

int main() {
    MyNamespace::greet();
    return 0;
}

This example defines a function inside a namespace and calls it from main().
