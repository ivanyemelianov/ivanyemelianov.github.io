Preprocessor --> Compiler --> Linker

Preprocessor
- Executes all #'s
- Results into a translation unit with all the includes pasted in

Compiler
- Checks syntax
- Creates .obj files from all .cpp files
- Object files are translations of code to machine language

Linker
- Combines all .obj files into a single .exe file
- Finds all unresolved symbols and replaces with their definitions
