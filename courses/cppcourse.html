Functions in C++ allow code reuse and modularity by grouping instructions into reusable blocks. A function takes input (parameters), performs an operation, and optionally returns a value.

#include // Function declaration int add(int a, int b); // Main function int main() { int sum = add(5, 3); std::cout << "Sum: " << sum << std::endl; return 0; } // Function definition int add(int a, int b) { return a + b; }
  • Return Type – Specifies the type of value returned (e.g., int, void).
  • Parameters – Input values passed to the function.
  • Declaration & Definition – Functions can be declared before use and defined later.
  • Function Call – Executes the function and retrieves the return value.

If statements in C++ are used to make decisions in a program based on a certain condition. The syntax for an if statement is as follows:

if (condition) { // code to execute if condition is true }

For example, the following code checks if the variable "x" is greater than 5 and if so, prints "x is greater than 5".

int x = 10; if (x > 5) { std::cout << "x is greater than 5" << std::endl; }

You can also include an else statement after an if statement to execute code if the condition is false. The syntax for an if-else statement is as follows:

if (condition) { // code to execute if condition is true } else { // code to execute if condition is false }

You can also use else if statement to check multiple conditions.

if (condition1) { // code to execute if condition1 is true } else if (condition2) { // code to execute if condition1 is false and condition2 is true } else { // code to execute if both condition1 and condition2 are false }

In C++, you can use an initialization statement in an if statement to declare and initialize a variable before the condition is evaluated. The variable declared in the initialization statement is only in scope within the if statement and is only visible in the block of the if statement. The syntax for an if statement with an initialization statement is as follows:

if (auto variable = expression; condition) { // code to execute if condition is true }

Here "variable" is the variable that is declared and initialized with the result of "expression", and "condition" is the condition that is evaluated to determine whether the code in the if statement should be executed. For example, the following code uses an initialization statement to declare and initialize a variable "x" with the result of the function "getValue()", and then checks if "x" is greater than 5:

if (auto x = getValue(); x > 5) { std::cout << "x is greater than 5" << std::endl; }

In C++, a for loop is used to repeatedly execute a block of code a certain number of times. The syntax for a for loop is as follows:

for (initialization; condition; increment) { // code to execute }

For example, the following code uses a for loop to print the numbers from 1 to 10:

for (int i = 1; i <= 10; ++i) { std::cout << i << std::endl; }

You can also use for loop for iterate through containers like vector, array, etc.

std::vector myVector = {1,2,3,4}; for(auto x : myVector) { std::cout << x << std::endl; }

This is called range-based for loop introduced in C++11, it allows you to iterate over elements of container.

In C++, a while loop is used to repeatedly execute a block of code as long as a certain condition is true. The syntax for a while loop is as follows:

while (condition) { // code to execute }

For example, the following code uses a while loop to print the numbers from 1 to 10:

int i = 1; while (i <= 10) { std::cout << i << std::endl; i++; }

In C++, a do-while loop is similar to a while loop, but the code inside the loop is executed at least once before the condition is evaluated. The syntax for a do-while loop is as follows:

do { // code to execute } while (condition);

For example, the following code uses a do-while loop to repeatedly ask the user to enter a number between 1 and 10 until they provide a valid input:

int num; do { std::cout << "Enter a number between 1 and 10: "; std::cin >> num; } while (num < 1 || num > 10);

In C++, the break and continue statements are used to control the flow of a loop. The break statement is used to exit a loop early, before the loop condition is met. When a break statement is encountered inside a loop, the loop is immediately terminated and the program continues to execute the code following the loop. For example, the following code uses a while loop to repeatedly ask the user to enter a number between 1 and 10. If the user enters 0, the loop is terminated using a break statement:

int num; while (true) { std::cout << "Enter a number between 1 and 10 or 0 to exit: "; std::cin >> num; if (num == 0) { break; } // more code here }

The continue statement is used to skip one iteration of a loop and move on to the next iteration. When a continue statement is encountered inside a loop, the remaining code inside the current iteration is skipped, and the next iteration of the loop begins. For example, the following code uses a for loop to print all the even numbers between 1 and 10, but skips the iteration if the number is odd:

for (int i = 1; i <= 10; i++) { if (i % 2 != 0) { continue; } std::cout << i << std::endl; }

In C++, the goto statement is used to transfer control to a labeled statement within the same function. A labeled statement is a statement that is preceded by an identifier followed by a colon.

goto label;

For example, the following code uses a goto statement to transfer control to the labeled statement "end" when the variable "x" is greater than 5:

int x = 10; if (x > 5) { goto end; } std::cout << "x is less than or equal to 5" << std::endl; end: std::cout << "This code is executed regardless of the value of x" << std::endl;
