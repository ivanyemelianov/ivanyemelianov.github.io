Wordpress theme created for a NFT collection called Beargorand.

This WordPress theme for NFT collections is built using the Underscores theme template, and is designed specifically for showcasing and managing non-fungible token (NFT) collections.

The theme features advanced filtering capabilities, allowing users to easily browse and search for specific NFTs based on various attributes such as rarity or NFT image features like background color, facial expresion, accesories, and more.

Additionally, the theme is fully responsive and optimized for fast loading and easy navigation, making it the perfect choice for anyone looking to create a professional and functional website for their NFT collection.

Main feature of this project is an advance filter system made in php
